## Homey CLI
```
$ npm install -g homey
```
```
$ homey login
$ homey list
```
```
$ homey app create
```
```
# app.json 경로에서 접근
$ node -v
$ npm install
$ homey app run
```

## OAuth2
버튼(Button): 일반 로그인 (ID/PW 직접 입력)
조명(Light Bulb): OAuth2 로그인 (GitHub 연동 등)
